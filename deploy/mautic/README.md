# Mautic Integration

Use Mautic for self-hosted marketing automation:

- landing forms
- email nurturing
- segmentation
- lead scoring
- consent-aware follow-up

Lead records must include:

- source content id
- campaign
- offer
- persona
- UTM fields
- consent flag
- qualification score
- next action

The Lead Router Agent may create or update leads, but it must not send emails unless the message is approved and consent rules pass.
